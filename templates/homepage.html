Welcome to lastNote

Your terminal-based task manager for ultimate productivity... or procrastination.

About lastNote

lastNote is the ultimate procrastinator-friendly task management app designed for your terminal. It ensures that every time you open your terminal, you're greeted with your tasks, motivating you to stay productive (or at least try to).

Features

  • Add Notes: Write down your big ideas or "I'll do this tomorrow" moments.
  • View All Notes: Remind yourself how productive you were (or weren’t).
  • View Today's Notes: Get called out on the tasks you’ve been avoiding today.
  • Delete Notes: Pretend they never existed.
  • Humorous Terminal Presence: No more escaping from your to-dos; they’ll greet you every time you open your terminal.

How It Works

Here's what makes lastNote unique:

  • Notes by ID: Tasks are saved with a unique ID in an SQLite database.
  • Smart Filters: Easily view today’s tasks or all tasks at once.
  • Persistent Storage: Notes are stored safely in a database, ensuring they’re always available (for better or worse).
  • Automatic Deletion: Outdated tasks are deleted automatically, keeping things tidy.

Installation

  1. Clone the GitHub repository: 
    git clone https://github.com/elfat-py/lastNote
  2. Navigate to the project folder: 
    cd lastNote
  3. Install dependencies: 
    pip install termcolor
  4. Set up automatic terminal execution:
    • For Windows: Copy the provided startTerminalWithNotes.bat file to your Startup folder: 
      shell:startup

Usage

Once installed, here’s how you can use lastNote:

  • Add Notes: Title your task and give it a body (or leave it blank for later).
  • View Notes: See all your notes listed by ID, date, title, and body.
  • View Today’s Notes: Only see tasks scheduled for today.
  • Delete Notes: Erase tasks you no longer need.

To run the app manually:

python main.py

Or let it run automatically each time you open your terminal (if you set up the startup file).

Pro Tips

  • Mess up a date? Just delete the note and start over.
  • Too many notes? Enjoy scrolling through your tasks… endlessly.
  • Afraid of commitment? Use the "Delete Note" option liberally—no judgment here!
